Anime Recommendation System

Bu proje Python kullanılarak geliştirilmiş basit bir anime öneri sistemidir.

Özellikler

Tür bazlı anime filtreleme
Rastgele anime önerisi (Surprise Me)
Kullanıcı arayüzü (ipywidgets)
Görsel destekli sonuçlar
Kullanılan Teknolojiler

Python
Pandas
ipywidgets
Amaç

Kullanıcıya seçtiği türe göre anime önerileri sunmak ve veri filtreleme mantığını göstermek.
